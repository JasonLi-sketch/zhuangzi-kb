## 关系类型

| 关系 | 含义 | 庄子示例 |
|------|------|----------|
| 体用关系 | 体→用：本体与发用 | 道→德，天→自然 |
| 同一关系 | 彼此等同或互含 | 逍遥=无待，齐物=两行 |
| 工夫关系 | 手段→境界：如何达成 | 心斋→逍遥，坐忘→吾丧我 |
| 对反关系 | 对立统一/超越 | 有用↔无用，天↔人，知↔不知 |
| 寓言关系 | 故事→寓意：寓言映射概念 | 庖丁解牛→养生，混沌开七窍→无为 |
